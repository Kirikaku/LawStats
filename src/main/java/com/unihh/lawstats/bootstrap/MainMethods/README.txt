Some of the features of this software are used only once in the entire curse of the software usage.
Generally they are never used by an enduser.
Therefore we did not create an additional userinterface to trigger these features.
In order to provide a convenient way of using these functionalities and also to give usage examples we provide the following classes.
Each class contains a main method that executes the functionality indicated by the class name.
You will find additional information in each class.
